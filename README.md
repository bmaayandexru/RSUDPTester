# RSUDPTester Программа для тетирования RS и UDP соединений

Программа может передавать пакеты по интерфейсам RS и Ethernet(UDP). 
Настройка интефейсов интуитивно понятна.
Содержимое пакета размещается в текстовом поле. Пакет предается при нажатии на
кнопку "Передать". Отправляемая и передаваемая информация отображается в окне 
"Трафик". Ведется статистика приема/передачи. 
Более подробная информация в файле RSUDPTester.pdf
Прорамма является базовой и может быть переделана под нужды потребителя. Предложения отправляйте на почту
