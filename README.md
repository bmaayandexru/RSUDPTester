#RSUDPTester Программа для тетирования RS и UDP соединений
Программа может передавать пакеты по интерфейсам RS и Ethrnet(UDP). 
Настройка интефейсов интуитивно понятна.
Содержимое пакета размещается в текстовом поле. Пакет предается при нажатии на кнопку "Передать". Отправляемая и передаваемая информация отображается в окне "Трафик". Ведется статистика приема/передачи. Прорамма является базовой и может быть переделана под нужды потребителя. Предложения отправляйте на почту bmaa@yandex.ru
